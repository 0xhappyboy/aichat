# 快捷键

## 通用快捷键

- `Enter` - 进入聊天/发送消息
- `F1` - 显示/隐藏帮助
- `C` - 切换为中文界面
- `E` - 切换为英文界面
- `1-4` - 切换主题
- `Q` - 退出应用

## 聊天界面导航

- `← →` - 切换 AI 模型
- `↑ ↓` - 滚动聊天历史
- `PageUp/Down` - 快速滚动
- `Home/End` - 跳至顶部/底部

## 编辑模式

- `i` - 进入编辑模式
- `Esc` - 退出编辑模式
- `Backspace` - 删除字符
- `Delete` - 清空输入
