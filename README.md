# HotKey

## General Hotkeys

- `Enter` - Start chat/Send message
- `F1` - Show/Hide help
- `C` - Switch to Chinese
- `E` - Switch to English
- `1-4` - Switch themes
- `Q` - Quit app

## Chat Navigation

- `← →` - Switch AI models
- `↑ ↓` - Scroll chat history
- `PageUp/Down` - Fast scroll
- `Home/End` - Jump to top/bottom

## Edit Mode

- `i` - Enter edit mode
- `Esc` - Exit edit mode
- `Backspace` - Delete character
- `Delete` - Clear input
