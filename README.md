# HotKey

## General

- Enter - Start chat / Send message
- F1 - Show / Hide help
- C - Switch to Chinese
- E - Switch to English
- 1-4 - Switch themes
- Q - Quit app

## Welcome Screen

- Enter - Start chatting
- C - Switch to Chinese
- E - Switch to English
- 1-4 - Switch theme
- F1 - Show help
- Q - Quit app

## Chat Navigation (Normal Mode)

- ← → - Switch AI models
- ↑ ↓ - Scroll chat history
- PageUp - Scroll up 10 lines
- PageDown - Scroll down 10 lines
- Home - Jump to top
- End - Jump to bottom

## Input Mode Controls

- i - Enter edit mode
- Esc - Exit edit mode (clear input)
- Enter - Send message and return to normal mode

## Edit Mode

- Enter - Send message and exit edit mode
- Esc - Exit edit mode without sending
- Char keys - Type text
- Backspace - Delete last character
- Delete - Clear entire input
